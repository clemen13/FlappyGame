FlappyGame

Juego flappybird desarrollado en Java como parte de aprendizaje de Java y Swing.

Características

- Jugabilidad fluida con física de gravedad y salto  
- Pipes generados aleatoriamente con espacio ajustable  
- Score y High Score visibles en pantalla  
- Reinicio al presionar **SPACE** después de Game Over

Tecnologías

- Java   
- Swing para renderizado 2D  
- Timers para loop y generación de pipes  
- Persistencia de High Score mediante archivo de texto  

---

Cómo jugar

1. Clona el repositorio:
   git clone https://github.com/clemen13/FlappyGame.git
   
2. Abre el proyecto en IntelliJ IDEA

3. Ejecuta GameFrame.java

4. Presiona SPACE para saltar y evitar pipes

Supera tu High Score
